# Befunge

This is quite a fast interpreter for [Befunge-93](https://en.wikipedia.org/wiki/Befunge).

## State

100% compliance with the Befunge-93 specifications.

## Performance

see benchmarks.txt

## How to build:

### Production:

```
make
```

### Development

```
make devel
```

## How to run

```
./befunge *my-befunge-program.file*
```
